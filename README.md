# GitHub Finder App

A GitHub user search application built with React, Vite, Context API, Tailwind CSS, and DaisyUI. Users can search for GitHub profiles and view user details and repositories. This project demonstrates React hooks, global state management with Context and useReducer, and working with an external API. This project was built as part of a React course.

## Features

- Search GitHub users by username  
- View individual user profiles  
- Display user repositories   

## Technologies Used

- **React (Vite)** – Frontend framework and build tool  
- **React Context API & useReducer** – State management  
- **React Router DOM** – Client-side routing  
- **Tailwind CSS** – Utility-first styling  
- **DaisyUI** – UI component library  
- **GitHub REST API** – External data source  
- **JavaScript (ES6+)** – Core language
